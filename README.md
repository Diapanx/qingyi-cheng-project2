# Minesweeper
A complete minesweeper game build with React and Node.js.

[Play Now!](https://qingyi-cheng-project2.onrender.com)

# Features
- Autoclear an empty cell
- Set flags
- Show mines left
- Reset the game
- Three levels of difficulty

# Build
Git clone this project and then

`npm run dev`
